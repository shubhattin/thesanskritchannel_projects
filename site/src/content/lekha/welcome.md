This is the built-in **Markdown** pipeline for the site. Posts live in `src/content/lekha/` as `.md` files; each file’s name becomes the URL slug under `/lekha/`.

## What you can do next

- Add more `.md` files in `src/content/lekha/`.
- Set `draft: true` in frontmatter to hide a post from the index (the post page can still be linked directly if you want to change that later).
- Later, you can swap the data source while keeping the same `/lekha/[slug]` URLs.

Happy writing.
